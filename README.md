# 📊 Customer Insights Platform

A Streamlit-powered analytics application designed to provide comprehensive e-commerce intelligence across customer segmentation, revenue metrics, sales forecasting, and inventory tracking.

---

## ✨ Features

* **Customer Report & Lifetime Value Ledger:** Tracks total order frequency, net customer spend, and active timelines.
* **Predictive & Machine Learning Engine:** Integrated modules for demand forecasting, churn prediction, and product recommendations.
* **Multi-Portal Architecture:** Features dedicated administrative views and vendor monitoring systems.
* **Interactive Data Visualizations:** Real-time metrics powered by Pandas and Parquet datasets.

---

## 🛠️ Tech Stack

* **Frontend/Dashboard:** Streamlit
* **Data Processing:** Pandas, PyArrow (Parquet)
* **Machine Learning:** Scikit-learn
* **Language:** Python 3.12+
